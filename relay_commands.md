# ESP32 继电器控制指令表

## 1. 单路独立控制
**格式**：4位字符 (前2位是继电器编号 `01-08`，后2位是动作 `01`开 / `00`关)
此指令只改变指定的继电器状态，不影响其他继电器。

| 继电器 | **开启指令** | **关闭指令** | 说明 |
| :--- | :--- | :--- | :--- |
| **第1路** | `0101` | `0100` | 控制 RELAY1 (GPIO 32) |
| **第2路** | `0201` | `0200` | 控制 RELAY2 (GPIO 33) |
| **第3路** | `0301` | `0300` | 控制 RELAY3 (GPIO 25) |
| **第4路** | `0401` | `0400` | 控制 RELAY4 (GPIO 26) |
| **第5路** | `0501` | `0500` | 控制 RELAY5 (GPIO 27) |
| **第6路** | `0601` | `0600` | 控制 RELAY6 (GPIO 14) |
| **第7路** | `0701` | `0700` | 控制 RELAY7 (GPIO 12) |
| **第8路** | `0801` | `0800` | 控制 RELAY8 (GPIO 13) |

---

## 2. 全量控制 (一次性覆盖所有8路状态)
**格式**：2位十六进制字符 (`00` ~ `FF`)
此指令会一次性设置所有8个继电器的开/关状态。
*每一位二进制对应一个继电器，1为开，0为关。*

| 功能 | **指令** | 说明 |
| :--- | :--- | :--- |
| **全开** | `FF` | 所有继电器吸合 (二进制 `11111111`) |
| **全关** | `00` | 所有继电器断开 (二进制 `00000000`) |
| **只开前4路** | `0F` | 1-4开，5-8关 (二进制 `00001111`) |
| **只开后4路** | `F0` | 5-8开，1-4关 (二进制 `11110000`) |
| **间隔开** | `AA` | 2,4,6,8开 (二进制 `10101010`) |
| **间隔开** | `55` | 1,3,5,7开 (二进制 `01010101`) |

## 3. 示例代码
### JavaScript (网页控制台)
```javascript
var ws = new WebSocket("ws://192.168.18.252:81");
ws.onopen = function() {
    console.log("Connected");
    ws.send("0101"); // 开启第1路
};
```
